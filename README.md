## 時計
システム時刻を表示

## 今日の天気
`Open-Meteo API`を使って天気情報を取得
デフォルトでは東京都に設定
郵便番号を入力して`変更`ボタンを押すことで、天気の場所を変更できる

## 週間予報
`Open-Meteo API`を使って天気情報を取得
デフォルトでは東京都に設定
郵便番号を入力して`変更`ボタンを押すことで、天気の場所を変更できる

## 運行情報
Yahoo路線情報(Yahoo路線情報 関東)を使って運行情報を表示

## 最新ニュース
yahooニュースRSS(Yahooニュース トピックス)を使ってニュース情報を表示

## カレンダー
Googleカレンダーの埋め込みコードで表示
